# AI-Driven Robotic Exploration Simulation

Academic project featuring a Python simulation for an autonomous robotic agent tasked with identifying a target in an unknown 2D space.

## Key Features
* Implemented evolutionary mechanisms (proportional selection, uniform crossover) from scratch.
* Designed a custom genetic diversity algorithm based on Hamming Distance to optimize the search trajectory and prevent premature convergence.

## Technologies Used
* Python
* NumPy & Matplotlib
* Genetic Algorithms Optimization
